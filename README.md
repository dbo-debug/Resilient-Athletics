# Resilient Athletics

Minimal Vite + React + TypeScript landing page for a training gym.

## Run locally

```bash
npm install
npm run dev
```

## Structure

```text
resilient-athletics/
├── index.html
├── package.json
├── src/
│   ├── App.tsx
│   ├── components/
│   │   ├── ButtonLink.tsx
│   │   ├── ImagePlaceholder.tsx
│   │   ├── SectionHeading.tsx
│   │   └── ServiceCard.tsx
│   ├── main.tsx
│   └── styles.css
├── tsconfig.app.json
├── tsconfig.json
├── tsconfig.node.json
└── vite.config.ts
```

Add the real logo to an assets folder later and replace the hero placeholder block as needed.
